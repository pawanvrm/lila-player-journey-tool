# Player Journey Visualization Tool

## Overview
This tool visualizes player movement and behavior using game telemetry data.

## Features
- Player movement on minimap
- Event markers (loot, kill, death)
- Playback over time
- Heatmap of player activity
- Filters (map, match, player)

## Tech Stack
- Streamlit
- DuckDB
- Plotly

## Run Locally
pip install -r requirements.txt
streamlit run app.py
